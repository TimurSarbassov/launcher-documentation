= Deploying {runtime} application to standalone Red Hat Enterprise Linux

:parent-context: {context}
To deploy your {runtime} application to standalone Red{nbsp}Hat Enterprise Linux, configure the `pom.xml` file in the application, package it using Maven and deploy using the `java -jar` command. 

.Prerequisites

* RHEL 7 or RHEL 8 installed.
